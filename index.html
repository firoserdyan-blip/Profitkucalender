ProfitKu: Aplikasi Kalender Profit & Loss Harian

Selamat datang di ProfitKu, aplikasi web (Single Page Application) yang dirancang untuk membantu trader, investor, dan pelaku bisnis online melacak riwayat keuangan harian mereka dengan mudah. Kami menggunakan **penyimpanan lokal (*local storage*) browser Anda** sehingga data hanya tersimpan di perangkat Anda. Data ini tidak dikirim ke server manapun, sehingga **privasi Anda terjamin 100%**. Jangan khawatir soal biaya server! *Peringatan: Jika Anda mengganti browser atau menghapus riwayat browser, data Anda akan hilang.*
